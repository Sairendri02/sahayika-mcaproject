{% trans "Sahayika System" %}

{% trans "Sahayika is a digital platform developed to simplify the management of Self Help Groups (SHGs). It replaces traditional manual record-keeping methods with a structured and efficient system where all financial and member-related data can be stored, accessed, and managed in one place." %}

{% trans "The main purpose of this system is to improve transparency, reduce errors, and save time in handling group activities such as savings collection, loan distribution, and repayment tracking. It ensures that every transaction is properly recorded and easily accessible when needed." %}

{% trans "How to Use the System" %}

{% trans "Users can log in to the system using their credentials. Based on their role, such as President or Member, they can perform different actions. The President can add members, record monthly collections, and manage loans, while members can view their savings, loan details, and payment status." %}

{% trans "To view past records, users can select a specific month and year in the monthly collection and loan tracking sections. This allows easy access to historical data without confusion." %}

{% trans "Rules and Guidelines" %}

{% trans "All entries must be recorded accurately at the time of the meeting to maintain correct financial data. Only authorized users should be allowed to add or modify records. Members should regularly check their data to ensure transparency and correctness." %}

{% trans "Loan repayments and savings contributions should be updated monthly. Any delay or incorrect entry may affect the overall group financial report. Proper usage of the system ensures fairness and accountability among all members." %}

{% trans "Benefits of Digital System" %}

{% trans "Using a digital system like Sahayika reduces dependency on paper records, minimizing the risk of data loss or damage. It also provides faster access to information, making decision-making easier for the group." %}

{% trans "The system improves accuracy in calculations, automatically tracks totals, and helps monitor loan repayments efficiently. It also enhances transparency, as every member can view their own financial status clearly." %}

{% trans "Conclusion" %}

{% trans "Sahayika is a reliable and user-friendly solution for managing SHG activities digitally. By adopting this system, groups can ensure better organization, improved financial control, and long-term sustainability." %}
